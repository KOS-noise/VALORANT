## 📝 Project Overview

This repository contains an early HTML-based clone of the official **VALORANT** website, specifically inspired by the Riot Games Korea web service. 

### 🛠️ Development Details

* **Authentic UI/UX:** All interactive elements, including button `onclick` events and navigation flows, were meticulously structured to replicate the genuine user experience of the official Riot Games website.
* **Custom Design Assets:** The visual assets—such as agent portraits, backgrounds, and character descriptions—were custom-designed from scratch and implemented as temporary mockups for this draft.
* **The 72-Hour Sprint:** This entire project was built and deployed within a highly focused 72-hour timeframe.
* **100% Hand-Coded:** Developed entirely before the era of AI coding assistants like ChatGPT. Every single line of HTML and CSS in this repository was typed by hand, reflecting pure dedication and raw coding effort.

> **Note:** This project serves as a personal archive of early front-end development practice, showcasing raw markup and styling skills without relying on modern AI generation tools.
